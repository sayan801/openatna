Welcome to OpenATNA.

OpenATNA is an implementation of an Audit Record Repository supporting RCF 3881 audit messages over BSD Syslog as well as RFC 5424-5426 (UDP and TLS).

Use the links on the left to download the source code from subversion and read the documentation.

OpenATNA is released under the Apache Version 2 Licence
